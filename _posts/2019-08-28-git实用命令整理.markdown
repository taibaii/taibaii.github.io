git实用命令整理  
  ================================
  ## 客户端新建分支并push到远端
  
  1. git remote -v
  
   查看当前路径资源所对应的远端地址，返回git路径，可用git clone拉取
  ***
  
  2. git checkout -b newbranch localbranch
  
  从本地已存在的localbranch切出新分支newbranch
  ***
  
  3.git push --set-upstream origin newbranch
  
  把本地新建的分支推送到远端
  ***
  
  4. git push origin --delete 远端分支名称（一般删除多余的remote分支）

   本地客户端删除远端分支
 ***
